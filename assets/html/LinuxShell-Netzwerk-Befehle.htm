Netzwerk-Schnittstellen Infos:

ifconfig, iwconfig & iwlist

ifconfig -a
Netzwerk-Adapter ein-/ausschalten
ifconfig wlan0 up # oder "down"
Drahtlose Netzwerk-Schnittstellen
iwconfig
WLAN-Zugangspunkte
iwlist scan

ping, arp -a, arp-scan & netstat

ping 8.8.8.8 Internetverbindung prüfen
arp -a zeigt verfügbare Netzwerkgeräte
arp-scan 192.168.0.0/24 zeigt verbundene Netzwerkgeräte
netstat -nano zeigt TCP/IP Netzwerkverbindungen

route -n, traceroute, mtr & tracepath

route -n zeigt lokale Routing-Tabelle
traceroute 8.8.8.8 Internet-Packetverfolgung
mtr traceroute-Altenative:
tracepath 8.8.8.8 Zeigt Gateway-Routen mittles ICMP TIME_EXCEEDED Anfragen. Traceroute-Alternative mittels MTU, UDP. In Debian mit apt-get install iputils-tracepath herunterladen.

tcpdump Tool für Netzwerk-Monitoring

(In Debian mit apt-get install tcpdump herunterladen.)
tcpdump -i eth0 -n net 192.168.0.1

iptables Port schließen

iptables -A INPUT -p tcp --dport 23 -m state --state NEW,ESTABLISHED -j DROP
Filterfunktion verbergen
iptables -A INPUT -p tcp --dport 23 -m state --state NEW,ESTABLISHED -j REJECT

service Netzwerk-Reset

service networking restart

systemctl Systemstart-Dienste konfigurieren

systemctl

ss "Socket Statistics"

Zeigt Kommunikations-Punkte
ss -a

wpa_supplicant Wlan Debugging

Wlan reconnecten:
wpa_supplicant -i wlan0 -D wext -c /etc/wpa_supplicant/wpa_supplicant.conf -dd
