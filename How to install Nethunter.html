How to install Kali Nethunter on Android Device (with Nexus Root Toolkit_v2.1.9):

backup all your data with 'adb backup –all -apk -shared –f c:\backup.ab' or use twrp (official twrp app)
Download device factory image.zip
Download TWRP Recovery Image
configure usb to MTP
copy downloaded files to device via usb
Press Buildnumber 7 times for Developer-Mode
Enable USB Debbuging
Enable OEM Unlock
restart device to bootloader (Hold Powerbutton + VolumeDown)
run Nexus Root Toolkit programm (NRT)
plug in device
select model type
NRT - Full Driver Installation Guide - Install Google Drivers (only)
NRT - Unlock Bootloader
restart device to bootloader (Hold Powerbutton + VolumeDown)
NRT - install factory rom + unroot
Press Buildnumber 7 times for Developer-Mode
Enable USB Debbuging
Install Official TWRP App
restart device to bootloader (Hold Powerbutton + VolumeDown)
NRT - root device
Press Buildnumber 7 times for Developer-Mode
Enable USB Debbuging
open SuperSU App allow permissions
open BusyBox App press 'Install'
Install Official TWRP App
Download nethunter.zip
Download TWRP Recovery Image
configure usb to MTP
copy downloaded files to device via usb
open TWRP and flash Image 
restart device to recovery mode (Hold Powerbutton + VolumeDown)
install nethunter.zip via twrp teamwin recovery mode
