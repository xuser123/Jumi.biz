ifconfig Netzwerk-Schnittstellen Infos:
ifconfig -a
Netzwerk-Adapter ein-/ausschalten
ifconfig wlan0 up # oder "down"
iwconfig Drahtlose Netzwerk-Schnittstellen zeigen
iwconfig
iwlist Suche nach Wifi-Zugänge
iwlist scan
ping Netzwerk Verbindungstest
Erreichbarkeit prüfen
ping 8.8.8.8 # oder www.microsoft.com
arp -a Verbundene Netzwerk-Geräte zeigen:
arp -a
arp-scan Netzwerk-Geräte suchen:
arp-scan 192.168.0.0/24
netstat Netzwerkverbindungen TCP/IP zeigen
netstat -nano
route -n Lokale Routing-Tabelle:
route -n
traceroute Internet-Packetverfolgung:
Zeigt Gateway-Routen mittles ICMP TIME_EXCEEDED Anfragen:
traceroute 8.8.8.8
tracepath traceroute-Alternative (mittels MTU, UDP):
In Debian mit apt-get install iputils-tracepath herunterladen.
tracepath 8.8.8.8
tcpdump Tool für Netzwerk-Monitoring
In Debian mit apt-get install tcpdump herunterladen.
tcpdump -i eth0 -n net 192.168.0.1
iptables Port schließen
iptables -A INPUT -p tcp --dport 23 -m state --state NEW,ESTABLISHED -j DROP
Filterfunktion verbergen
iptables -A INPUT -p tcp --dport 23 -m state --state NEW,ESTABLISHED -j REJECT
service Netzwerk-Reset
service networking restart
systemctl Systemstart-Dienste konfigurieren
systemctl
ss Socket Statistics
Zeigt "Kommunikations-Punkte"
ss -a
