Linux Shell Befehle

cd Verzeichnis wechseln
Bsp. $ cd /home/user/Desktop
ls Verzeichnisinhalt auflisten
Bsp. $ ls /home/user/Desktop
dir -a Auch versteckte Dateien auflisten

pwd Aktuellen Verzechnis-Pfad zeigen

rm Dateien/Ordner entfernen
Bsp. $ rm /home/user/Desktop/text.txt
mv Dateien/Ordner verschieben/umbennen
Bsp. $ mv /home/user/Desktop/text.txt /home/user/Documents/
cp Dateien/Ordner kopieren
Bsp. $ cp /home/user/Desktop/text.txt /home/user/Documents/copy.txt
Ordner rekursiv kopieren:
Bsp. $ cp -r /home/user/Desktop/folder /home/user/Documents/copy
