Linux Shell Befehle

cd Change Directory
cd /home/user/Desktop
Verzeichnis wechseln

ls List Directory
ls /home/user/Desktop
Verzeichnisinhalt auflisten

dir -a Directory -all
cd -a /home/user/Desktop
Auch versteckte Dateien auflisten

pwd Print Working Directory
pwd /home/user/Desktop/text.txt
Aktuellen Verzechnis-Pfad zeigen

rm Remove
rm /home/user/Desktop/text.txt
Dateien/Ordner entfernen

mv Move
mv /home/user/Desktop/text.txt /home/user/Documents/text.txt
Dateien/Ordner verschieben/umbennen

cp Copy
cp /home/user/Desktop/text.txt /home/user/Documents/copy.txt
Dateien/Ordner kopieren

cp -r Copy -recursive
cp -r /home/user/Desktop/folder /home/user/Documents/copy
Rekursiv kopieren (z.B. Ordnerinhalt)

mkdir Make Directory
cp /home/user/Desktop/text.txt /home/user/Documents/copy.txt
Ordner/Verzeichnis erstellen

rmdir Remove Directory
rmdir /home/user/Desktop/Folder
Ordner/Verzeichnis löschen

sudo Superuser Do
sudo rm /home/user/Desktop/text.txt
Befehl als Superuser ausführen

passwd Password
passwd user
Passwort ändern

chmod Change Modification
chmod +x /home/user/Desktop/text.txt
Datei-Berechtigung ändern

whoami Who am I
whoami
Aktuellen Benutzer anzeigen

man Manual
man whoami
Anleitung zu einem Befehl Anzeigen

locate Find files by name
locate sudoers
Dateien über Dateinamen finden

cat Concatenate files and print
cat /home/user/Desktop/text.txt
Dateiinhalt ausgeben / einfügen
